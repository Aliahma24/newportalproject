ATTENDANCE SYSTEM - DET AILED FLOW
How It Works
Teacher  Side:
1. Teacher logs in
2. Goes to "Mark Attendance" section
3. Sees today's classes with scheduled times
4. "Mark Present" button becomes active at class start time (e.g., 08:00 PM)
5. Teacher clicks "Mark Present" when class starts
6. System records timestamp
Timing Rules:
On Time:  Marked present between scheduled time and +5 minutes
Late:  Marked present after 5 minutes but before end of class = Late/Penalty recorded
Absent:  If teacher never clicks button = Automatically marked absent
Student Side:
Student also has "Mark Present" button on their side
Both teacher AND student marking attendance provides double verification
If only teacher marks but student doesn't = Flag for review
If only student marks but teacher doesn't = Teacher absent
UI Design Notes for  Attendance
Button states:
Disabled (gray) - before class time
Active (green) - during class time window
Late (orange) - after 5 minutes
Missed (red) - after class ended
Timer countdown showing time until class starts
Auto-refresh to enable buttons at correct time
